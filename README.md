# Brainfuck interpreter in Go

## Feature
- REPL
- Executable String
- Runable File
- Context
- Easy to read the code

## Quick Start
```
go get github.com/janqx/brainfuck-go
```

## Simple Example
```bash
git clone https://github.com/janqx/brainfuck-go.git
 
cd brainfuck-go

# REPL
go run cli/main.go
>> ++++++++++++++++++++++++++++++++++++
>> .
$  
>> +.
%  
>> !exit

# Execute String
go run cli/main.go -c "++++++++[>++++[>++>+++>+++>+<<<<-]>+>+>->>+[<]<-]>>.>---.+++++++..+++.>>.<-.<.+++.------.--------.>>+.>++."
Output: Hello World!
```
